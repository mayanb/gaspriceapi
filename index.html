Gas Price API

This api returns a list of gas stations and their prices near you

Go to https://gas-price-api.herokuapp.com/stations/[zipcode] where [zipcode] is filled in with the zipcode you are searching

For example to get gas prices in Palo Alto, CA you would go to https://gas-price-api.herokuapp.com/stations/94305

The api returns a JSON object called stations within which each nearby gas station is listed

Each gas station object contains the name of the station (name), the price (price), and the address (address).
